# QSPIO-CRC: Quantitative Systems Pharmacology Platform for Immuno-Oncology in Colorectal Cancer Treatment Using T cell Engager and Anti-PD_L1

This repository contains the latest QSP platform for CRC developed by the Popel Systems Biology Laboratory https://popellab.johnshopkins.edu/

## Requirements
- Model simulations and analyses were performed using MATLAB R2020b 
- SimBiology Toolbox (Model Structure and Simulation)
- Statistics and Machine Learning Toolbox (Sensitivity Analysis)
- Bioinformatics Toolbox (Visualization)
- Symbolic Math Toolbox (Units and Dimensional Analysis / Model Initialization)
